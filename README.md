# JavaScript Drum Kit 🥁

An interactive drum kit web application built using HTML, CSS, and JavaScript. 

## Features
- Mouse click sound detection
- Keyboard key event handling
- Dynamic sound playback
- Button press animations
- Event-driven JavaScript logic

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation & Event Listeners)

## How to Use
Press the corresponding keys or click the drum buttons to play sounds.
